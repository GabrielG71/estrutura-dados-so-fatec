## Estrutura de Dados

Disciplina pratica com implementacoes em C cobrindo estruturas de dados sequenciais, lineares e nao-lineares, com enfase em gerenciamento de memoria dinamica e complexidade de operacoes.

### Atividades

| Atividade | Tema                                | Conceitos Abordados                                       |
|-----------|-------------------------------------|-----------------------------------------------------------|
| AA01      | Sistema de menu interativo          | Funcoes, fluxo de controle, entrada/saida                |
| AA02      | Calculadora de media e variancia    | Vetores, ponteiros, calculo estatistico                   |
| AA03      | Simulacao de sorteio (loteria)      | Geracao aleatoria, deteccao de duplicatas                 |
| AA04      | Gerenciamento de estoque            | Structs aninhadas, controle de inventario                 |
| AA05      | Calculo de potencia recursivo       | Recursao, caso base, aritmetica com inteiros longos       |
| AA06      | Lista encadeada simples             | Alocacao dinamica (malloc/free), insercao, remocao, busca |
| AA07      | Fila (FIFO)                         | TAD Fila, disciplina FIFO, operacoes O(1) nas extremidades|
| AA08      | Pilha (LIFO)                        | TAD Pilha, disciplina LIFO, strings com buffer fixo       |
| AA09      | Lista duplamente encadeada circular | Estrutura circular, travessia bidirecional, menu interativo|

### Tecnologias

- **Linguagem:** C (padrao C99)
- **Compilador:** GCC
- **Plataforma:** Windows / Linux

## Sistemas Operacionais

Disciplina teorica com atividades dissertativas cobrindo os principais conceitos de sistemas operacionais modernos, desde fundamentos ate problemas classicos de sincronizacao e deadlocks.

### Atividades

| Atividade | Tema                                          |
|-----------|-----------------------------------------------|
| AA01      | Fundamentos de SO e instalacao de sistemas    |
| AA02      | Mainframes e supercomputadores                |
| AA03      | Chamadas de sistema (system calls)            |
| AA04      | Virtualizacao e emulacao                      |
| AA05      | Interfaces de linha de comando (CLI)          |
| AA06      | Processos, componentes e escalonamento        |
| AA07      | Threads, sincronizacao e modelos de thread    |
| AA08      | Condicoes de corrida e exclusao mutua         |
| AA09      | Problemas classicos: Jantar dos Filosofos e Leitores-Escritores |
| AA10      | Deadlocks — condicoes necessarias e estrategias de tratamento   |
